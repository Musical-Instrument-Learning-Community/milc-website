<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
* npm
  ```sh
  npm install -g yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Musical-Instrument-Learning-Community/milc-website.git
   ```
2. Install yarn packages
   ```sh
   yarn install
   ```
3. Run
   ```sh
   yarn start
   ```
